# 交互

# prompt

prompt 接受两个参数，result = prompt(title, [default])

  • title：显示给用户的文本
  • default：可选的参数，指定input框默认的初始值

如果访问者在输入框中输入信息，则result 返回值返回输入的内容，如果用户点击取消/ESC，则result 返回null

# confirm

confirm 函数显示一个带有question 以及确定和取消两个按钮的模态窗口。

点击确定返回true ，点击取消返回false 。
