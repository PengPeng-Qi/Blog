# 游戏
狭义上的游戏是**通过游戏引擎制作的电子游戏**。  
  
游戏的分类：
- ACT：动作类
- AVG：冒险类
- 模拟类
- RPG：角色扮演
- SLG：策略类
  
游戏的开发：策划、程序、美术。  
  
游戏引擎最大的优势：渲染。  
## 游戏引擎
- `The NVL Maker` -- 文字冒险游戏制作
- `RPG Maker` -- 关卡类型游戏

## Web游戏引擎
利用`Canvas` 和`WebGL` 为底层技术抽象的图像绘制库。  
  
- `Cocos`：3D能力 不成熟
- `Laya`：支持JS、TS、3D能力比较成熟
- `Egret`：国产
- `CreateJS & Phaser`：没有可视化界面

功能引擎：
- 2d渲染引擎：Pixi.js
- 3d渲染引擎：Three.js (WebVR)
- 2d物理引擎：Box2D.js
