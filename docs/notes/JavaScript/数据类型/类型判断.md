## 总结
- 1、`instanceof` 可以准确地判断复杂引用数据类型，但是不能正确判断基础数据类型；  
- 2、`typeof` 它虽然可以判断基础数据类型（null 除外），但是引用数据类型中，除了`function` 类型以外，其他的也无法判断。  

> `typeof` 是一元运算符
