# 基本数据类型
## null
`Null` 特值**对象的值未设置**
### undefined与null的区别
```js
undefined === null;  // false
undefined == null;   // true

null === null;       // true
null == null;        // true

Number(null);        // 0
Number(undefined);   // NAN

isNaN(1 + null)      // false
isNaN(1 + undefined) // true
```
> 注意：`==` 与`===` 前者会进行类型转换
## undefined
声明没定义的变量的初始值  
  
`undefined` 是全局对象的一个属性，也就是说，它是全局作用域的一个变量。  
  
`undefiend` 属性的属性特征：
| undefined属性的属性特征 |       |
| ----------------------- | ----- |
| writable                | false |
| enumerable              | false |
| configurable            | false |
  
> `undefiend` 可能在非全局作用域中被当作标识符来使用，因为`undefiend` 不是一个**保留字**
  
要**检查一个变量是否在全局上下文中存在可以通过检查全局对象上是否存在这个属性**
```js
if ('x' in window) {
  // 只有x被全局性的定义 才会这行这些语句
}
```
## BigInt
`BigInt` 是通过在整数末尾附加`n` 或**调用构造函数**来创建的。  
  
`BigInt` 不能与数字互换操作。否则，将抛出`TypeError`。
