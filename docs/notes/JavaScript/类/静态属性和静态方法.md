# 静态属性和静态方法
我们可以把一个方法赋值给类的函数本身，而不是赋值给他的`prototype`。这样的方法被称为**静态的(static)**  
  
在一个类中，我们以`static` 关键字开头，如下：
```js {2}
class User {
  static staticMethod() {
    alert(this === User);
  }
}

User.staticMethod(); // true
```
实际上以直接将其作为属性赋值的作用相同：
```js
class User { }

User.staticMethod = function() {
  alert(this === User);
};

User.staticMethod(); // true
```
**静态方法用于实现该类但不属于该类任何特定对象的函数**。
## 静态属性
前面加有`static`:
```js
class Article {
  static publisher = "Levi Ding";
}

alert( Article.publisher ); // Levi Ding
```
等同于直接给`Article` 赋值：
```js
Article.publisher = "Levi Ding";
```
注意区分**类字段**：
```js
class Article {
  publisher = "Levi Ding";
}

let arc = new Article();
console.log(arc.publisher);  // Levi Ding
```
## 继承静态属性和方法
**静态属性和方法是可以被继承的。**  
  
`Animal.compare` 和`Animal.planet` 是可被继承的，可以通过`Rabbit.compare` 和`Rabbit.planet` 来访问：  
```js {2,14-16,32,36}
class Animal {
  static planet = "Earth";

  constructor(name, speed) {
    this.speed = speed;
    this.name = name;
  }

  run(speed = 0) {
    this.speed += speed;
    alert(`${this.name} runs with speed ${this.speed}.`);
  }

  static compare(animalA, animalB) {
    return animalA.speed - animalB.speed;
  }

}

// 继承于 Animal
class Rabbit extends Animal {
  hide() {
    alert(`${this.name} hides!`);
  }
}

let rabbits = [
  new Rabbit("White Rabbit", 10),
  new Rabbit("Black Rabbit", 5)
];

rabbits.sort(Rabbit.compare);

rabbits[0].run(); // Black Rabbit runs with speed 5.

alert(Rabbit.planet); // Earth
```
**`extends` 让`Rabbit` 的`[[Prototype]]` 指向了`Animal`**。  
  
`Rabbit extends Animal` 创建了两个`[[Prototype]]` 引用：  
  
- `Rabbit` 函数原型继承自`Animal` 函数。  
- `Rabbit.prototype` 原型继承自`Animal.prototype`。  

```js
class Animal {}
class Rabbit extends Animal {}

// 对于静态的
alert(Rabbit.__proto__ === Animal); // true

// 对于常规方法
alert(Rabbit.prototype.__proto__ === Animal.prototype); // true
```
## 其他
`class Rabbit {}`: 
```js
Rabbit.__proto__ === Function.prototype;
Rabbit.prototype.__proto__ === Object.prototype;

// error，Rabbit 中没有这样的函数
alert ( Rabbit.getOwnPropertyNames({a: 1, b: 2})); // Error
```
> `Object.getOwnPropertyNames(obj)`方法返回一个由指定对象的所有自身属性的属性名
  
`class Rabbit extends Object {}`:
```js
Rabbit.__proto__ === Object;
Rabbit.prototype.__proto__ === Object.prototype;
```
