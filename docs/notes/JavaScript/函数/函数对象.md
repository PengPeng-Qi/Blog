# 函数对象 NFE
## 属性 name
函数对象包含一些便于使用的属性，例如，一个函数的名字可以通过属性`name` 来访问：
```js
function sayHi(){
  console.log('Hi')
}

console.log(sayHi.name);
```
对象的方法也有名字：
```js
let user = {

  sayHi() {
    // ...
  },

  sayBye: function() {
    // ...
  }

}

alert(user.sayHi.name); // sayHi
alert(user.sayBye.name); // sayBye
```
无法推测名字的时候，此时，属性`name` 会是空的。
```js
// 函数是在数组中创建的
let arr = [function() {}];

console.log( arr[0].name ); // <空字符串>
// 引擎无法设置正确的名字，所以没有值
```
## 属性 length
返回函数参数的个数：
```js
function f1(a) {}
function f2(a, b) {}
function many(a, b, ...more) {}

alert(f1.length); // 1
alert(f2.length); // 2
alert(many.length); // 2
```
`rest` 参数不参与计数。  
## 自定义属性
我们也可以添加自己的属性，在这里添加`counter` 属性，用来跟踪总的调用次数：
```js
function sayHi() {
  alert("Hi");

  sayHi.counter++;
}
sayHi.counter = 0; // 初始值

sayHi(); // Hi
sayHi(); // Hi

alert( `Called ${sayHi.counter} times` ); // Called 2 times
```
**属性不是变量，比如`sayHi.counter = 0`，不会在函数内定义一个局部变量`counter`**。属性`counter` 和变量`let counter` 是毫不相关的两个东西。  
  
函数属性会被用来替代闭包，使用函数属性将变量作用域
```js
function makeCounter() {
  // 不需要这个了
  // let count = 0

  function counter() {
    return counter.count++;
  };

  counter.count = 0;

  return counter;
}

let counter = makeCounter();
alert( counter() ); // 0
alert( counter() ); // 1
```
**这样，`counter` 被直接存在函数里，而不是他外部的词法环境。**  
```js
function makeCounter() {

  function counter() {
    return counter.count++;
  };

  counter.count = 0;

  return counter;
}

let counter = makeCounter();

counter.count = 10;  // 只有嵌套函数可以访问
console.log( counter() );  // 10
```
## 命名函数表达式
命名函数表达式(`NFE`)，指带有名字的函数表达式。
```js {1}
let sayHi = function func(who) {
  console.log(`Hello, ${who}`);
}
```
关于名字`func` 有两个特殊的地方，这就是添加它的原因：
1. 它允许函数在内部引用自己
2. 它在函数外是不可见的

```js
let sayHi = function func(who) {
  if (who) {
    alert(`Hello, ${who}`);
  } else {
    func("Guest"); // 使用 func 再次调用函数自身
  }
};

sayHi(); // Hello, Guest

// 但这不工作：
func(); // Error, func is not defined（在函数外不可见）
```
这时候有一个疑问？我们为什么不直接使用`sayHi` 进行嵌套调用？  
  
因为`sayHi` 的值可能会被函数外部的代码改变。如果函数赋值给另一个变量，那么函数就会开始报错：
```js
let sayHi = function(who) {
  if (who) {
    alert(`Hello, ${who}`);
  } else {
    sayHi("Guest"); // Error: sayHi is not a function
  }
};

let welcome = sayHi;
sayHi = null;

welcome(); // Error，嵌套调用 sayHi 不再有效！
```
该函数从它的外部词法环境获取`sayHi`。没有局部的`sayHi` 了，所以使用外部变量，而当调用的时候，外部的`sayHi` 是`null`。  
  
可以通过给函数表达式添加可选的名字解决此类问题。
```js
let sayHi = function func(who) {
  if (who) {
    alert(`Hello, ${who}`);
  } else {
    func("Guest"); // 现在一切正常
  }
};

let welcome = sayHi;
sayHi = null;

welcome(); // Hello, Guest（嵌套调用有效）
```
`func` 是函数局部域的，不是从外部获取的。
