# 选择符
## 类选择器
### 多个类
```css
p.warning.help {
  background-color: red;
}
```
匹配`class` 属性同时包含`warning` 和`help` 的p 元素
```html
<p class='urgent warning help'>Help Me!</p>
```
## 属性选择器
### 简单属性选择器
若想让**同时具有`href` 和`title` 属性**的HTML 超链接显示为粗体，可以这样写：
```css
a[href][title] {
  font-weight: bold;
}
```
### 根据精准的属性值选择
只选择属性为特殊值的时候进行匹配：
```css
a[href='https://pengpeng-qi.github.io/Blog/'] {
  font-weight: bold;
}
```
### 根据部分属性值选择
| 形式              | 说明                                                                            |
| ----------------- | ------------------------------------------------------------------------------- |
| `[foo|='bar']`    | 选择的元素有foo属性，且其值以bar 和一个英文破折号开头，或者值就是bar 本身       |
| `[foo~='bar']`    | 选择的元素有foo属性，且其值是一组以空格分隔的词中包含bar 这个词或只有bar 这个词 |
| `[foo*='bar']`    | 选择的元素有foo属性，且其值包含子串bar                                          |
| `[foo^='bar']`    | 选择的元素有foo属性，且其值以bar开头                                            |
| `[foo$='bar']`    | 选择的元素有foo属性，且其值以bar结尾                                            |
| `[foo$='.pdf' i]` | 选择的元素有foo属性，且其值以bar结尾，**匹配属性值不区分大小写**                |

## 根据文档结构选择
| 形式    | 说明                                                |
| ------- | --------------------------------------------------- |
| h1 + p  | 紧跟在h1 元素后面的所有p 元素，且二者同属一个父元素 |
| h2 ~ ol | 选择一个同属于一个父元素的另一个元素，不用紧跟      |
