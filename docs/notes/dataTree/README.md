# 数据结构与算法

数据结构就是一组数据的存储结构，算法就是操作数据的一些方法。

常见的十个数据结构：数组、链表、栈、队列、散列表、二叉树、堆、跳表、图、Trie 树。

常见的十个算法：递归、排序、二分查找、搜索、哈希算法、贪心算法、分治算法、回溯算法、动态规划、字符串匹配算法。

## 复杂度

大 O 时间复杂度并不表示具体代码真正的执行时间，而是表示**代码执行时间随数据规模增长的变化趋势**，所以，也叫做**渐进时间复杂度**，简称**时间复杂度**。
